### Graph Data Structure
Graph由顶点以及边所组成, 这里只考虑无向图, 并且没有自回路.

#### 一些记号
单独的顶点, 我们用$v_1$, $v_2$, ..., $v_n$来表示.
如果$v_1$, $v_2$之间存在边, 我们用<$v_1, v_2$>来表示.

#### 概括(几个文件的描述)

**graph.h**文件包含了数据结构`Graph<T>`的定义. **operations_on_graph.h**中包含了常见的对于图的操作, 比如从用户以特定格式读取一个图. **graph_algorithms.h**文件中包含了类`GraphAlgorithm`, 一些对图的常用算法组成了这个类的静态成员函数, 比如得到给定连通图的最小生成树. **test.cpp**文件利用Boost.Test对编写的代码进行测试. **main.cpp**可以写一些应用, 暂时还没有实施.
